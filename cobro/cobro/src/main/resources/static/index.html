Bienvenido

Bienvenido al sistema de gestión de préstamos de Inversiones Homero. Esta plataforma fue desarrollada para facilitar la administración de clientes, préstamos, cobros, pagos, gastos, reportes y cierres de caja, permitiendo llevar un control seguro y organizado de toda la información financiera.

¿Qué puedes hacer?

  • Administrar clientes.
  • Registrar nuevos préstamos.
  • Registrar pagos y abonos.
  • Consultar el historial de cada cliente.
  • Ver préstamos activos y finalizados.
  • Gestionar gastos del negocio.
  • Realizar cierres de caja.
  • Consultar indicadores financieros.
  • Generar reportes.
  • Revisar alertas de cobro.
  • Administrar usuarios (solo administradores).

Recomendaciones de uso

  • Verifica que estés utilizando tu usuario asignado.
  • No compartas tu contraseña.
  • Registra todos los pagos inmediatamente después de recibirlos.
  • Registra cualquier gasto realizado durante el día.
  • Revisa las alertas de clientes con cuotas vencidas.
  • Realiza el cierre de caja al finalizar la jornada.

Buenas prácticas y Seguridad

  • No elimines registros sin autorización.
  • Verifica los datos antes de guardar.
  • Mantén actualizada la información de los clientes.
  • Adjunta comprobantes cuando sea necesario.
  • Justifica cualquier gasto o diferencia de caja.
  • No compartas tu usuario.
  • Cierra sesión cuando termines de trabajar.
  • Evita dejar la aplicación abierta sin supervisión.
  • Reporta cualquier comportamiento extraño al administrador.

Soporte

Si presentas algún inconveniente o duda con el sistema, por favor contacta directamente al administrador del sistema.

Manual de Uso y Guía Rápida de Módulos

1. Módulo Clientes

Registro y búsqueda: Permite ingresar clientes con su DNI, datos de contacto, dirección física y zona/barrio. Puedes subir una foto de perfil en Base64.

Perfil de Cliente: Muestra un resumen financiero completo de los créditos otorgados, abonos realizados, saldo restante y estado de mora.

2. Módulo Préstamos

Cálculo Automático: Ingresa el monto solicitado y el interés (ej: 20%). El sistema genera de forma instantánea el valor por cuota y la fecha final.

Estados: Los créditos pueden estar ACTIVO, PAGADO o ATRASADO si existen cuotas impagadas en base a la fecha.

3. Módulo Registrar Pago

Cobro Diario: Selecciona el préstamo del listado lateral, ingresa la cantidad recaudada y el método (Efectivo/Nequi).

Recibo Imprimible: Al guardar el cobro, se genera una plantilla de ticket lista para descargar en formato de texto plano y enviar por WhatsApp.

4. Módulo de Gastos

Egresos Operativos: Administra los costos del negocio (Transporte, Nómina, Alimentación, etc.).

Obligatorio: No se permite guardar ningún gasto sin escribir una justificación clara. Puedes adjuntar fotos de comprobantes o facturas.

5. Módulo Cierre de Caja

Flujo diario: El sistema calcula el saldo en caja esperado sumando el capital e intereses cobrados en el día y restando los gastos registrados. Introduce el dinero físico que tienes contado.

Control de descuadres: Si el dinero no coincide con el esperado, el sistema exige justificar detalladamente si se trata de un sobrante o faltante antes de permitir registrar el cierre.
